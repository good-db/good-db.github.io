    Class GoodDB

    The main class for the GoodDB package

    Example: Using JSONDriver (sync)

    const db = new GoodDB(new JSONDriver({ path: './database.json' }));

    Example: Using MongoDBDriver (async)

    const db = new GoodDB(new MongoDBDriver({ uri: "..." }));
    await db.connect();
